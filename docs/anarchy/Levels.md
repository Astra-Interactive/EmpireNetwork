Уровень игрока будет виден перед его ником в табе и над головой. У каждого уровня будет свой цвет. Цены за прокачку
добавлю после того, как будет готова и протестирована экономика. Денежное вознаграждение за привязку дискорда будет
равно цене за прокачку 1 уровня. Также, можно добавить такое же вознаграждение за голосование за сервер на мониторинге.
Для Каждого уровня ещё нужен будет свой пермишион крафта. По типу craftinglever[Уровень]
